# MIC-1-simulator
1. Instale o Docker na máquina se não tiver

2. Para encapsular o programa use o comando no terminal na pasta onde está esse arquivo txt para gerar a imagem do Simulador:
docker build -t mic1 .

3. Para rodar a imagem gerada:
docker run -p 8000:8000 mic1

4. Acesse no navegador http://localhost:8000/
